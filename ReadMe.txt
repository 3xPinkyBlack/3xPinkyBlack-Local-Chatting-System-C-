Thanks For Installing This Application

To Use Our Application With Out Recompiling It
You Have To Have Another Computer In Your Local Network/ The Computer You Are Gonna Use As a Server in Your Local network

1. Its IP Address Must Be 172.16.26.114 or Modify in The Source Code
2. Then Create A User Named kiot in The Server Computer With Password 1 or Modify the Source Code To Use What You Want
3. Share Documents Folder in The Created User kiot
4. Then Create A Folder Named chatFiles in The Documents Folder 

Thats it Now You Can You The Application as Admin or User Mode

You Can Get The Admin Binary in The Installation Folder/Windows-Binary/admin.exe
You Can Get The User App Binary in The Installation Folder/Windows-Binary/userApp.exe

You Can Modify And Compile The Source Code As You Want
